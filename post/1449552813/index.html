记录本人2024年看过的一些番剧

前言

写这篇 blog 的主要原因是停课在机房坐牢过于无聊，采取这种方式来消耗时间，同时也方便记录自己的入宅经理和感想。

对于记录的每一部番，我不仅会记录感想尽管有些忘得差不多了，同时也会从画质水准，剧情质量，内容深度，心里反馈以及等方面进行极端主观的评价和总评，不喜勿喷。

接下来记录大致按照时间顺序记录，这样更有意义。

早期

直到今年三月以前，我是很少看番的，除了国产的刺客伍六七和我的三体，日番我也只看过 JOJO 的前四部，此阶段我选择的番都是有原因的。

龙与虎

算不上远古老番，但是也挺早了，找到这部番属实不易，其实是因为它的一首歌才知晓的，然后就不知不觉地看完了。一部很平常的校园日常恋爱番，主角五人都有个性，剧情没有突出优点，画质由于远古所以一般，唯一不理解的就是大河凭什么赢得过小实（真的惨）和亚美啊。

  • 画质水准：★★★★★★☆☆☆☆
  • 剧情质量：★★★★★★★☆☆☆
  • 内容深度：★★★★★★☆☆☆☆
  • 心里反馈：★★★★★★★★☆☆（别问，问就是第一次看这类型的番产生了戒断反应，花了好几天才好起来）
  • 总评：7.57.57.5 分

赛博朋克：边缘行者

如果说《龙与虎》是因为音乐，这一部则是因为游戏，为了玩《赛博朋克：2077》特地看的一部番，虽然只有十集（好像就看了两天），但是网飞的确是把大卫这个形象和成为传奇到覆灭的故事叙述完整了，尤其是动画的高潮部分，动画和配乐都十分贴合。

虽然是一部刀子，但这就是赛博朋克世界观下的必然结果，一部资本主义悲剧，在那个世界是理所应当的。

  • 画质水准：★★★★★★★★☆☆
  • 剧情质量：★★★★★★★★★☆
  • 内容深度：★★★★★★★★★☆
  • 心里反馈：★★★★★★★★★☆（别问，问就是第一次吃刀子）
  • 总评：999​ 分

你的名字

神作。几年前第一次看的时候甚至没太看懂（时空穿越系是这样的），今年又重温了一下，观感好了许多，其实也是因为音乐好听才看的~~（最近沉迷日语歌）~~，今年暑假又奇迹般地重映了，去电影院又看了一遍。不得不说，这一部可以算作我心目中的最佳电影了，不论是从画质（全是电脑壁纸）还是音乐（不必多说，恰到好处的震撼人心）都是顶级的。

  • 画质水准：★★★★★★★★★★
  • 剧情质量：★★★★★★★★★★
  • 内容深度：★★★★★★★★☆☆
  • 心里反馈：★★★★★★★★★☆
  • 总评：9.59.59.5​​ 分

天气之子

神作，但这部今年是第一次看。同样也是新海诚灾难片三部曲之一。

与《你的名字》立意差不多，但故事的叙述明显更加现实忧郁，161616 岁离家出走的少年在东京卑微地活下去，不像上一部那样乌托邦式的美好结局，这一布反映了未成年人在社会上的无能为力。结局刻画也有所不同，帆高为了阳菜放弃了整个世界，有人可能觉得这过于自私，但我觉得这对于遭受社会冷落的少年唯一的选择。

  • 画质水准：★★★★★★★★★★
  • 剧情质量：★★★★★★★★★★
  • 内容深度：★★★★★★★★☆☆
  • 心里反馈：★★★★★★★★☆☆
  • 总评：9.59.59.5​​​ 分

秒速五厘米

听歌找过来的同样是一部电影，只有六十多分钟，由三个短篇构成，分别讲述了重逢与离别、单恋与思念、长大后的生活。一部名副其实的 BE 电影，但是篇幅太短导致第一次看连人名都没认清楚就结束了，更别说感动了（大抵只有少部分人才能引起共鸣吧）。但是那一首《One more time, One more chance》可谓是越听越难受。最近又重温了一遍电影，感触明显更深了。

这一部无疑是新海诚电影里最痛的一部，但也是最现实的一部。

  • 画质水准：★★★★★★☆☆☆☆（年龄比我大的电影）
  • 剧情质量：★★★★★★★★☆☆（要怪只能怪电影篇幅太短了却讲了三个故事）
  • 内容深度：★★★★★★★★★★
  • 心里反馈：★★★★★★★★★☆（看了你就知道了）
  • 总评：999​​​​ 分

中期

Charlotte

暑假看的第一部番（好像 B 站短视频刷到片段就追下去了）。一共 131313 集的由 Key 社和 Aniplex 联合企划的季番。这部番优点很多，缺点也多。看到前 666 集以为是纯日常番，完全不知道番名是什么意思。后面突然就给我刀出来了，不愧是 Key 社，不愧是麻枝准。尤其是最后面几集，从妹妹死了乙坂有宇黑化，听首歌就恢复记忆了，然后无中生哥，回溯时光，拯救世界……然后由刀一波大的，轮到哥哥黑化后，乙坂有宇决定拯救世界，剧情主打就是一个跳跃。

本来背景这么庞大我是完全可以接受的，但是它只做了 131313​ 集，合着拯救世界放到了最后一集，回来了就完结了。乙坂有宇恢复记忆的事是一点没写，糖是一点不发的，不看漫画结局真容易当成刀子。

也是这部番让我入坑了后摇。

  • 画质水准：★★★★★★★☆☆☆
  • 剧情质量：★★★★★★☆☆☆☆（四分给日常，两分给友利奈绪）
  • 内容深度：★★★★★★★★☆☆（好歹刻画了有宇的成长，从自私到被拯救到拯救）
  • 心里反馈：★★★★★★★★★☆
  • 总评：888​​​​ 分

我的青春恋爱物语果然有问题

慕名而来。

一般看别的番时，刷“大老师”的弹幕是最多的，因此找到了这部番。

本来打算分三季来讲的，第一部纯日常，第二部开始胃疼了，第三步硬是又疼了 101010​ 集（党争番忍不了一点）。

这本小说（不是番）的最大特点就是人物的性格塑造以及心理活动很细，而且人物对话很深，我在看的时候感觉一直在做阅读理解，稍微不留神就不知道他们在干什么了（感谢弹幕和评论解释），动画也尽量采用了大老师内心独白。不得不说，每一个看过春物的私宅都会说自己像大老师，唯一要骂就得骂做第二、三季的 feel. 社刻意给团子加戏份，让我的胃更痛了。团子在别的番高低得是女主，但这里她就不应该出现。

虽然但是，大老师的人格魅力以及至理名言永不过时（然后我就把四大校霸全看完了）。

  • 画质水准：★★★★★★★★☆☆（第一季一般，后两季有经费就是不一样）
  • 剧情质量：★★★★★★★★☆☆（999 分给渡航，扣一分给 feel. 社）
  • 内容深度：★★★★★★★★★☆（每句话都有深度）
  • 心里反馈：★★★★★★★★★☆
  • 总评：999​​​​ 分（就爱看日常校园恋爱番）

不时轻声地以俄语遮羞的邻座艾莉同学

唯一真史，慕名而来。

暑假最后悔的事就是 777 月晚上 22:3022:3022:30 按下了追番键。但是这次不怪动画工房，这次怪小说，我是真不知道小说为什么这么火。

首先是画质，经费有钱就是随便烧，画师ももこ也给力，每集的特效，OP 和 ED 都很好，这无话可说。

但是剧情就是一坨，拍个 121212 集一群人在学校过家家，美名其曰竞选学生会长，甚至第一季还没完，每一集都忍受着煎熬，最后几集甚至开两倍速。

人物塑造也是依托，男主无所不能，女主跟个废物一样，男主妹妹塑造得太好了，女主除了傲娇就是回忆杀，懒得骂。

  • 画质水准：★★★★★★★★★★
  • 剧情质量：★★☆☆☆☆☆☆☆☆（无话可说）
  • 内容深度：★★☆☆☆☆☆☆☆☆（无话可说）
  • 心里反馈：★☆☆☆☆☆☆☆☆☆（无话可说）
  • 总评：333​​​​​ 分（一分给画师，一分给 OP，一分给 ED）

甚至有第二季

我的三体第四季

今年国产新番，超出了我的预料。记得前两季经费约等于没有，第三季的《章北海传》就到达了很高的还原度，第四季也不赖。

基本上就是《三体Ⅲ》的前半本书，程心成为执剑人光速下班结束（999 集还是太少了），但是作画十分精湛，只有人是方形的，其他景象都是神中神的。而且这一季的运镜，剧情的展开明显更加成熟，至少让原著党满足了，还有很多原著中美发现的细节也交代清楚了。（吹爆的是最后一集程心面临抉择的 777 分钟的二维动画，顶中顶，着实把小说难以展现的内容描述出来了）。

原本《三体Ⅲ》的情感基调是偏向悲壮沉寂的，动画化中确实极力展现，但当你把立场转换到三体人身上，你不仅可以发现人类的愚昧、软弱与无知，甚至可以当爽文来看。

  • 画质水准：★★★★★★★★★☆
  • 剧情质量：★★★★★★★★★☆（进步远超预期）
  • 内容深度：★★★★★★★★☆☆（原著就摆在那）
  • 心里反馈：★★★★★★★★☆☆
  • 总评：999​​​​ 分（期待下一部）

ATRI - My Dear Moments

这里只评价游戏，不评价番剧。一是番还没完结，二是我看不下去了。

接触到这款游戏主要是因为 CG 精美，主要剧情不算新颖，很多 Gal 或者番都是机器人与人相恋然后机器人死 / 失去记忆的。这部游戏好就好在它的剧情铺垫，通过两波反转加上各种暗示强化作用，从而展现出一种完全不同的观感。

接下来大量剧透：具体来说，就是剧本写的好。一开始的 ATRI 已经说明了是机器人，但前期通过不断打破这个设定，让玩家误以为” ATRI 是有真心的，是真人而不是机器人“这一冲突观念，接着在看日志那里直接反转，将这一冲突观念直接推向顶峰，达到一种十分震撼的反转。同时这一转折也给玩家一种”ATRI 不存在真心，一切都是伪造的“的观念，而这一观念随着贯穿着整个故事的暗线——几十年前的故事再一次发生了转折。也就是 HE 开头，一切谜底全部解开。

如果你只是看上面一段，你可能没什么感想，但是这游戏就厉害在当你不看剧透时，它带来的震撼时无与伦比的。

  • 画质水准：★★★★★★★★★☆（插画是真的可以）
  • 剧情质量：★★★★★★★★★★
  • 内容深度：★★★★★★★★☆☆（其实也是对机器人伦理的一种探讨）
  • 心里反馈：★★★★★★★★★★
  • 总评：9.59.59.5​​​​ 分（这里还是说游戏）

青春猪头少年系列

慕名而来。

小说到现在还没有完结，番剧还在不停的做，目前就看了一部季番和一部剧场版（后面的应该都是剧场版了）。

本来以为是日常校园恋爱番，结果既不日常也不校园，全文贯穿量子力学，而且每一个故事都是单独分开的，又感人的也有偏搞笑的。

剧情没什么好说的，因为全文不贯穿的，但是每个故事都很新颖精彩，再加上师傅的个人魅力，不得不说是部好番（剩下的等我补完再说）。

  • 画质水准：★★★★★★★★★☆
  • 剧情质量：★★★★★★★★☆☆
  • 内容深度：★★★★★★★☆☆☆
  • 心里反馈：★★★★★★★☆☆☆（虽然但是也有感人的）
  • 总评：999​​​​ 分

埃罗芒阿老师

当成过渡番看的，纯纯的日常番。

感觉后面会变成后宫党争番，但是第一季没做出来，小说也完结了确定了正宫，姑且当作纯爱番。

  • 画质水准：★★★★★★☆☆☆☆
  • 剧情质量：★★★★★★☆☆☆☆
  • 内容深度：★★★☆☆☆☆☆☆☆
  • 心里反馈：★★★★☆☆☆☆☆☆
  • 总评：555​​​​ 分（没啥好评价的）

冰菓

原名好像不是这个，但是仍然慕名而来。

是一部几乎没有恋爱元素的校园推理番就好这口，番一共做了前四卷内容，总的来说剧情完全没有起伏，全靠对话的形式推动故事发展，但也有几个震撼人心的节点比如“冰菓”名字的由来，十文字事件的真相。看这类型的番重要的就是跟着进行推理，然后得出真相的瞬间，然后继续下去。所以这种推理番看了就停不下来。

  • 画质水准：★★★★★★★★★☆（京阿尼制作没得说）
  • 剧情质量：★★★★★★★☆☆☆
  • 内容深度：★★★★★★★☆☆☆
  • 心里反馈：★★★★★★★☆☆☆
  • 总评：8.58.58.5​​​​​ 分

中二病也要谈恋爱 1 / 2 / TOM

慕名而来，但是没想到这番有点抽象。

第一季纯日常番，勉强看得下去，第二部画质好了，但是又开始党争 +++ 恋爱，忍着看完的。

怎么说呢，除了看六花以外没有任何值得看的点，可谓是一人拯救一部番。而且男主也不好评价，女主一直在发电，总的来说观感一般。

  • 画质水准：★★★★★★★★☆☆
  • 剧情质量：★★★★★☆☆☆☆☆
  • 内容深度：★★★☆☆☆☆☆☆☆
  • 心里反馈：★★★★☆☆☆☆☆☆
  • 总评：555​​​​​​ 分（不好这口）

欢迎来到实力至上主义教室 1 / 2 / 3

也是一部推理 / 智斗的校园番至少前三季是这样的，这个故事的设定都是非常有意思的，对于我来说也是那种看了就停不下来的系列，三季一周不到就看完了。毕竟路哥设定就是天花板的，一年级篇纯纯就是爽文。

但是看完番再看小说就会发现其实番的路哥人设不是塑造的很完美，小说里面几乎一般都是路哥的心里独白，而动画中路哥的心理独白几乎没有，其实他也是有小头的，只不过他一直在利用人。番的风评差还有一点就是改的太多，甚至有些改的不能逻辑自洽，但是小说还是很不错的，现在还在连载三年级篇。

明年就要出第四季力。

  • 画质水准：★★★★★★☆☆☆☆
  • 剧情质量：★★★★★★★★☆☆（动画很一般）
  • 内容深度：★★★★★★★☆☆☆
  • 心里反馈：★★★★★★★★☆☆
  • 总评：888​​​​​​​ 分（就好这口）

辉夜大小姐想让我告白 1 / 2 / 3 / 剧场版

看的第一部恋爱喜剧番，没有主要剧情，没有内容深度，采用故事短篇的形式，每一集由两三个片段组成，但是一直在我的笑点上，看这部番就没停笑过，强推。

第四季会出的吧，会有的吧

  • 画质水准：★★★★★★★★☆☆
  • 剧情质量：★★★★★★★★★☆
  • 内容深度：★★★☆☆☆☆☆☆☆
  • 心里反馈：★★★★★★★★★☆
  • 总评：8.58.58.5​​​​​​​ 分（看过最轻松的一部番）

玉子市场 / 玉子爱情故事

最日常 +++ 最纯爱的一部番。

动画版只要就是女主的日常生活（挺喜欢那只鸡的），描绘了一个乌托邦的世界，看得挺困的。然后剧场版男主发力了，然后就没了。日常剧情也不能要求太高，我看到这的唯一感受就是加上《冰菓》和《中二病》，不愧是京阿尼，单看人脸都是分不出区别的。

  • 画质水准：★★★★★★★★★☆
  • 剧情质量：★★★★★☆☆☆☆☆
  • 内容深度：★★☆☆☆☆☆☆☆☆
  • 心里反馈：★★★★★☆☆☆☆☆
  • 总评：555​​​​​​​ 分（最无聊的一部）

樱花庄的宠物女孩

拜谢普通平衡树的第一篇题解，让我找到了这部不冷门也不知名的番。

一部真实深刻的成长番，男主和女二败犬都是普通人，而女主还有樱花庄的其他人都是他人难以企及的天才，男主的情感线也从远离、追求、绝望、嫉妒到最后的希望，忽略恋爱元素（本来就没多少）还是十分感人而贴切生活的一部番。唯一该骂的就是男主太亚撒西，可惜七海在别的番高低是个女主，还有万恶节操社弃坑。

  • 画质水准：★★★★★★★★☆☆
  • 剧情质量：★★★★★★★★★☆（感觉真的每一集都是电影的结尾）
  • 内容深度：★★★★★★★★★☆（从普通人的角度反应与天才的差距，可以说是《冰菓》的男二视角）

魔禁期

致敬传奇小说家镰池和马，致敬传奇小说系列《魔法禁书目录》，致敬传奇小说衍生系列《某科学的超电磁炮》、《某科学的一方通行》……，致敬传奇男主上条当麻。

简单解释一下，《魔法禁书目录》系列从 200420042004 年 444 月 101010 日开始出版，但现在还在连载。光是小说本体就写了606060多册（最近一次更新是在 999 月 101010 日），其中还不包括同样受欢迎的超炮系列等衍生作品。

可能你不知道是什么概念，但是节操社耗费大量精力，做了三季半年番（差不多 707070 多集），也只勉强做完了旧约（差不多二十本），过了二十年，和马仍然在以惊人的速度出版，节操社都快弃坑了。

整个魔法禁书目录这个大 IP，可以说入坑了就出不来了。三季半年番本体 + 三季半年番超炮 + 一季番科方 + 剧场版一共一百五十多集，我花了一个多月才全部追完。看番顺序如下：

某科学的超电磁炮、某科学的超电磁炮 S、魔法禁书目录、魔法禁书目录第二季、某科学的一方通行、某科学的超电磁炮 T、魔法禁书目录第三季、剧场版。

整个故事称得上有生之年类型的，但是也有主线（只不过推的很慢），相比之下，科学侧的支线明显更加受欢迎。这也导致了各个角色百花齐放，炮姐人气远超路人女主茵蒂克丝，上琴党很满意。故事不能说很新颖，但一直看下去不会反胃。节操社也尽量做到多支线不乱（要说的就是第三季剧情写的一团糟，暗部大战之类的起因等完全没有介绍，而且这一季的剧情反转就太公式化了，基本上就是反派装 B，教主说教，反派急眼，教主上去两拳，反派人格修正）。

而超炮这里是单线，所以动画化的剧情观感要更舒服，也把一些魔禁忽略的给补上了。但是科方做的真的是依托，动画偷工减料，女主人设也崩完了。
