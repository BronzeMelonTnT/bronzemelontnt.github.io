【题解】CF1931D Divisible Pairs

发表于2024-02-19|更新于2024-02-23
|字数总计:302|阅读时长:1分钟|阅读量:

解题思路

先看到第一个条件 ai+aj≡0 (mod x)a_i + a_j \equiv 0\ ({\rm mod}\ x)ai​+aj​≡0 (mod x)，可以推出如下性质：

ai mod x+aj mod x≡0 (mod x)a_i \ {\rm mod} \ x + a_j \ {\rm mod} \ x \equiv 0 \ ({\rm mod} \ x) ai​ mod x+aj​ mod x≡0 (mod x)

这样我们就可以在范围为 0∼x−10 \sim x - 10∼x−1 中查找满足条件的值，及对于每一个 aia_iai​，符合条件的 aja_jaj​ 为

(x−(ai mod x)) mod x(x - (a_i \ \rm mod \ x)) \ \rm mod \ x (x−(ai​ mod x)) mod x

而对于条件 ai−aj≡0 (mod y)a_i - a_j \equiv 0\ ({\rm mod}\ y)ai​−aj​≡0 (mod y)，只需要让 aia_iai​ 和 aja_jaj​ 同余即可。即：

ai≡aj (mod y)a_i \equiv a_j \ (\rm mod \ y) ai​≡aj​ (mod y)

我们只需要用 map 存储每个元素的两个条件，从而达到近似为 O(1)O(1)O(1) 的查询。
