【题解】CF1322B Present

发表于2024-05-11|更新于2024-10-25

Statement

给定一个长度为 nnn 的数列 aaa，求

⊕i=1n⊕j=i+1n(ai+aj)\oplus_{i = 1}^{n} \oplus_{j = i + 1}^{n} (a_i + a_j) ⊕i=1n​⊕j=i+1n​(ai​+aj​)

其中 1≤n≤4×1051 \le n \le 4 \times 10^51≤n≤4×105，∀i,1≤i≤n,1≤ai≤107\forall i,1 \le i \le n,1 \le a_i \le 10^7∀i,1≤i≤n,1≤ai​≤107。

Solution

简单的双重暴力显然是会超时的，问题的关键在于如何将两重循环减少一层。

对于异或等位运算操作，其特殊的性质在于按位运算，即每一位的影响不会改变下一层。

因此不难想到将答案二进制后的每一位进行讨论，最后统计答案。

  • 对于第 kkk 位：

    讨论两个数之和的二进制表示在第 kkk 位上是否为 111，根据加法的性质，我们只需要得知这两个数的前 kkk 位是多少。

    定义数组 bbb，对于 ∀bi\forall b_i∀bi​，都有 bi←ai∧(2k+1−1)b_i \gets a_i \wedge (2 ^ {k + 1} - 1)bi​←ai​∧(2k+1−1)。

    接着，我们讨论两个数（不妨设为 bx,byb_x, b_ybx​,by​）可行的条件：

    由于对于数组 bbb，(bx+by)(b_x + b_y)(bx​+by​) 的范围为 [0,2×(2i+1−1)][0,2 \times (2^{i + 1} - 1)][0,2×(2i+1−1)]，因此两个数可能会有两种情况：

    1. 在第 kkk 位不进位：bx+by∈[2k,2i+1−1]b_x + b_y \in [2^k, 2 ^ {i + 1} - 1]bx​+by​∈[2k,2i+1−1]​。
    2. 在第 k 位最多只进一位：bx+by∈[2×2k,2×(2i+1−1)]b_x + b_y \in [2 \times 2^k, 2 \times (2^{i + 1} - 1)]bx​+by​∈[2×2k,2×(2i+1−1)]。

​ 通过统计这两种方案的个数，根据异或的性质，当且仅当总方案为奇数时，说明 1≤i≤n,i+1≤j≤n1 \le i \le n, i + 1 \le j \le n1≤i≤n,i+1≤j≤n 对于第 kkk 位的贡献为 111，即 ansansans 在第 kkk 位上的二进制数为 111。

问题就在于如何以较低的复杂度求出满足一定范围的 bbb 数组元素的个数。

可以先将数组 bbb 进行排序，求范围个数不难想到双指针，定义两个指针分别指向满足左/右边界的最值，相减即为答案。由于序列 b 的单调性，当其中一个数单调上升，另外一个数成立的范围必然单调非递减，不妨让循环元素倒序遍历，这样就可以保证答案的正确性。

tips:tips:tips: 两个元素从两头遍历，答案会被统计两次，因此要除以 222。

总的时间复杂度为 O(nlog⁡nlog⁡w)\mathcal{O} (n \log n \log w)O(nlognlogw)，其中 nnn 为元素个数，www 为值域。
